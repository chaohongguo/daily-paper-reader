## 摘要
视角变化不变性和动作时序一致性是在未修剪视频中有效部署人体动作检测的关键方面。现有基于外观的视频检测方法在训练中常面临视角多样性有限的问题，而基于运动的检测方法则难以建模连续运动窗口间的细粒度时序关系。本文提出了一种新颖的两阶段动作检测方法，旨在提升视角不变性和全局时序连贯性。在第一阶段，我们仅于训练时从增强的虚拟视角中提取运动特征。第二阶段则引入了一种基于选择性状态空间序列建模的新型视角不变多尺度时序编码器，以聚合跨视角和时间尺度的信息。在PKU-MMD和BABEL基准上的实验表明，该方法在所有考虑的分割下均显著优于最先进的方法。代码和训练模型可在https://icb-vision-ai.github.io/HydraView-TAD获取。

## Abstract
Viewpoint change invariance and action temporal consistency are critical aspects for the effective deployment of human action detection of untrimmed videos. Existing appearance-based video detection methods often struggle with limited viewpoint diversity during training, while motion-based detection approaches frequently fail to model fine-grained temporal relationships across consecutive motion windows. This paper introduces a novel two-stage action detection approach designed to improve both view-invariance and global temporal coherence properties. In the first stage, we extract motion features from augmented virtual viewpoints, solely used at training. Then, the second stage introduces a new view-invariant, multi-scale temporal encoder based on selective state-space sequence modelling to aggregate information across viewpoints and time scales. Experiments on PKU-MMD and BABEL benchmarks demonstrate that this approach significantly outperforms state-of-the-art methods in all considered splits. Code and trained models are available at: https://icb-vision-ai.github.io/HydraView-TAD

## 一、研究动机与核心问题

该论文旨在解决**未修剪视频中人体动作检测**的两个关键瓶颈：

- **视角变化不变性**：训练时视角多样性有限，导致模型对同一动作在不同视角下的表现不一致，检测性能下降。
- **时序一致性建模不足**：现有基于运动（骨架）的方法通常只对全序列做全局池化，忽略了相邻时间窗口间的细粒度时序依赖，难以精确定位动作边界并保持时间上的连贯性。

整体而言，论文提出了一种名为 **HydraView-TAD** 的两阶段框架，**在训练时利用虚拟多视角数据增强特征表达**，并设计**基于选择性状态空间序列模型的多尺度时序编码器**，联合提升动作检测的视角鲁棒性和时序一致性。

## 二、方法论

### 2.1 总体架构
该方法采用**两阶段训练-推理范式**（推理时仅需单视角输入）：
- **第一阶段**：将输入视频滑动切分为固定长度的时间窗口（`F=16` 帧），通过虚拟相机渲染多视角 2D 骨架序列，利用专门设计的**时空图卷积网络（SWGCN）**提取窗口级运动特征；训练后冻结该编码器。
- **第二阶段**：将所有窗口的特征向量拼接成多视角特征张量，输入 **HydraView 多视角多尺度时序编码器**，输出每帧的多标签动作预测。

### 2.2 第一阶段：窗口特征运动编码
- **虚拟多视角生成**：
  训练时以 3D 运动序列为输入，通过 `V=12` 个虚拟相机（身体方位间隔 30°）投影得到多组 2D 骨架，并模拟自遮挡（根据躯干平面判断关节是否可见），增强视角多样性。
- **轻量时空图卷积网络（SWGCN）**：
  相比标准 ST-GCN，SWGCN 做了以下适配：
  - 取消时间降采样，保留精细帧级动作线索；
  - 调整时间卷积核尺寸以匹配短窗口（16 帧）；
  - 用 Group Normalization 替代 Batch Normalization，适应小批次训练。
  该编码器以多视角窗口作为输入，通过交叉熵损失监督训练后 **权重冻结**，为每个窗口输出一个特征向量 \(M^{v_i}_{a}\)。

### 2.3 第二阶段：视角不变多尺度时序编码器（HydraView）
**HydraView** 由多个并行的 **ViewMamba 模块**和一个**多尺度融合器**组成，整体对序列长度具有线性复杂度。

- **ViewMamba 模块（多尺度扫描）**：
  每个模块处理一种时间尺度 \(s\)，具体流程为：
  1. **2D 卷积投影**：沿视角维度以步长 \(s_v\) 做卷积，压缩视图数量、保留时间分辨率。
  2. **时序扩张**：以扩张因子 \(s_t = s\) 进行采样，扩大时间感受野。
  3. **选择性状态空间扫描（SSM）**：对展开后的序列执行**双向 2D 扫描**——沿视角维度正向/反向、沿时间维度左→右/右→左。通过输入依赖的线性参数 \(B_t, C_t, \Delta_t\) 动态建模跨视角交互与长时依赖（公式 2–4）。
  
- **多尺度融合器**：
  将所有尺度模块的输出拼合，用一层 **Vision Mamba**（在尺度维度上进行双向扫描）融合多尺度信息，最终通过 MLP 得到每帧的动作类别概率（使用二元交叉熵损失训练）。

## 六、主要结论与发现
- 在 PKU-MMD 两个分割上，HydraView 在 IoU 50% 时分别达到 **80.9%（CS）** 和 **94.5%（CV）**，显著高于 LAC、USDRL 等。
- 在 BABEL 三子集的 IoU 10% 指标上，相对强基线 LAC 提升 **+15%** 以上，在 IoU 50% 时也获得大幅增长（如子集 3 从 20.2 → 54.5）。
- 视角不变性（如“指向”动作）和时序连贯性（如“行走”动作）在定性与定量结果中均有明显改善。
- 多视角训练是核心增益来源之一；HydraView 的状态空间扫描策略比单纯多尺度卷积或普通 Mamba 变体更有效。

## 七、优点与亮点
- **问题洞察精准**：首次将视角不变性与时序一致性联合建模，并以两阶段训练分离窗口级运动表征与全局时序推理。

- **计算友好**：整个时序编码器为线性复杂度，推理速度快。
