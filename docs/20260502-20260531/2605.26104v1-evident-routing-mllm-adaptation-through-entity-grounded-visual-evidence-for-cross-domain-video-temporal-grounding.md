## 摘要
针对视频时间定位（VTG）微调多模态大语言模型（MLLMs）通常能提升域内性能，但在域偏移下性能急剧下降。本研究发现，这种失败主要不是由未见过的查询概念引起，而是由视觉域偏移导致，这阻碍了模型将其学习到的时间定位知识与其固有的实体注意力能力相结合。为此，我们提出EVIDENT，一个参数高效的适配框架，通过显式的视觉实体证据来路由VTG适配，将时间定位锚定在预训练MLLMs固有的实体注意力上。EVIDENT包含三个组件：(i) 实体瓶颈适配器，将密集的视觉令牌转化为紧凑的实体级槽位；(ii) 实体绑定蒸馏损失，向语义上非结构化的MLLM视觉空间注入物体性先验，引导每个槽位绑定到连贯的实体上；(iii) 实体到证据的门控机制，利用捕获的实体作为证据，引导模型定位包含查询相关实体的片段。这些组件共同使VTG微调能够依赖实体锚定的证据，而非脆弱的数据集捷径。在跨域VTG基准上的实验表明，EVIDENT在保持有竞争力的域内性能的同时，持续提升域外鲁棒性，且参数开销适中。这些结果表明实体级锚定是可泛化时间定位的有效归纳偏置。

## Abstract
Fine-tuning MLLMs for Video Temporal Grounding (VTG) often improves in-domain performance but degrades sharply under domain shift. In this work, we find that this failure is primarily driven not just by unseen query concepts, but by visual domain shift, which prevents the model from coupling its learned temporal localization knowledge with its inherent entity-attention capability. To address this, we introduce EVIDENT, a parameter-efficient adaptation framework that anchors temporal grounding in the inherent entity-attention of pre-trained MLLMs by routing VTG adaptation through explicit visual entity evidence. EVIDENT consists of three components: (i) an Entity Bottleneck Adapter that transforms dense visual tokens into compact entity-level slots, (ii) an Entity-Binding Distillation loss that instills objectness priors into the semantically unstructured MLLM visual space, guiding each slot to bind to a coherent entity, and (iii) an Entity-to-eVidence gating mechanism that leverages the captured entities as evidence, steering the model to localize moments containing query-relevant entities. Together, these components enable VTG fine-tuning to rely on entity-grounded evidence rather than brittle dataset shortcuts. Experiments on cross-domain VTG benchmarks show that EVIDENT consistently improves out-of-domain robustness while preserving competitive in-domain performance with modest parameter overhead. These results suggest that entity-level grounding is an effective inductive bias for generalizable temporal localization.

### 1. 论文的核心问题与整体含义
- **核心问题**: 将多模态大语言模型（MLLMs）微调用于视频时间定位（VTG）时，模型在源域（In-Domain, ID）性能提升，但在域外（Out-of-Domain, OOD）泛化能力急剧下降。传统观点常归因于未见过的查询概念，但本研究通过系统分析揭示 **视觉域偏移** 才是主要元凶——它导致模型习得的时间定位知识无法与预训练阶段本已具备的、与域无关的实体注意力（entity-attention）有效耦合，称为“注意力-定位解耦”（attention-localization decoupling）。
- **整体含义**: 要使 MLLM 在 VTG 中实现跨域鲁棒性，微调不能仅陷入源域的视觉上下文捷径，而应**将时间定位锚定在模型固有的实体注意力能力上**，通过显式的视觉实体证据来引导预测。为此，论文提出参数高效的 EVIDENT 框架，将定位过程路由至与查询相关的实体视觉证据，从而在几乎不破坏预训练对齐能力的前提下获得更强的域外泛化能力。

### 2. 方法论
EVIDENT 框架包含三个协同工作的组件，均以预训练 MLLM 的实体注意力为核心，并避免大规模指令微调。

- **实体瓶颈适配器 (EB Adapter)**
  - **目标**: 将 LLM 解码器早期层的密集视觉 token 重组为紧凑的**实体级槽位（entity-level slots）**，获得与域无关的视觉表示。
  - **实现**: 
    - 在 LLM 解码器的第 0–10 层插入轻量适配器。输入序列先经过下投影（down projection）降维，再对每一帧独立执行**槽位注意力（Slot Attention Block）**——可学习槽位查询与帧内视觉 token 进行竞争性注意力计算，并通过门控循环单元（GRU）迭代更新（共 3 次迭代），以鼓励每个槽位绑定到不同的语义实体。
    - 之后通过**重用槽位注意力权重进行 token 重构**，将原始密集 token 序列投影回原维度并与原始输入残差相加，从而实现实体级分解但不丢失序列长度。
  - **插入位置**: 仅在解码器早期层（0–10），利用该阶段跨帧交互来学习时间上连贯的实体解耦；深层（11–27）仅用 LoRA 微调，避免过度拟合源域。

- **实体绑定蒸馏 (EB Distillation)**
  - **动机**: 裸 MLLM 的视觉空间语义纠缠，即使有瓶颈结构，槽位也难以自发形成有意义的实体绑定（注意力分布近乎均匀）。
  - **实现**: 利用自监督模型 DINOv2 的 patch 特征，对每帧独立做 K-Means 聚类（K 设为槽位数），得到伪实体簇；通过匈牙利匹配将聚类结果与槽位注意力图进行一对一配对，再用二元交叉熵损失（BCE）**将 DINOv2 的对象性先验蒸馏到槽位注意力中**。该损失引导每个槽位聚焦于一个语义连贯的实体区域。

- **实体到证据门控 (E2V Gating)**
  - **目标**: 将捕获到的实体信息转化为**指示目标时刻的显式证据**，注入“目标时刻是查询相关实体共同出现的片段”这一归纳偏置。
  - **实现**: 
    - 用离线 LLM 提取查询中的主体（Subject）和客体（Object）概念词，并获取其在 LLM 输入序列中对应的 token 特征。
    - 对每一帧，计算帧平均视觉 token 与主体/客体查询 token 的余弦相似度，分别做 min‑max 归一化后**相乘得到帧级共现得分**。
    - 用该得分广播至该帧所有重构视觉 token，**调制 EB Adapter 输出的残差项**，使适配器只在实体共同存在的帧上施加显著影响。
- **训练目标**: 总损失 = 语言建模交叉熵损失 + λ · EB 蒸馏损失，端到端更新 EB Adapter 和 LoRA 参数（视觉编码器冻结）。

### 6. 主要结论与发现
- 微调 MLLM 在 VTG 中出现的 OOD 退化**主要由视觉域偏移引发**，而非查询概念未见。视觉域偏移使习得的定位知识无法转移到域外视频。
- 预训练 MLLM 本身具备强大的域无关实体注意力，但标准微调过程破坏了这种能力，导致“注意力‑定位解耦”。
- EVIDENT 通过 **EB Adapter + EB 蒸馏 + E2V 门控** 将时间定位路由至查询相关实体的视觉证据，在极低参数量下**显著提升 OOD 鲁棒性**，同时保持甚至略超过 ID 性能。例如，Charades→QVH 上 R1@0.5 的 OOD 提升 5.5 点，R1@0.7 提升 7.2 点。
- 实体级视觉表示作为归纳偏置，可作为一种有效的方式增强 MLLM 在 VTG 任务上的跨域泛化能力。

### 7. 优点
- **问题洞察深刻**: 精准识别视觉域偏移为 OOD 失败的元凶，并通过概念、视觉双重拆解与噪声干扰实验进行严谨验证。
- **方法优雅且参数高效**: 不重新训练视觉‑语言对齐，仅增加 2.36% 可训练参数，适合实际部署。
- **组件设计环环相扣**: EB Adapter 实现实体解耦，EB 蒸馏注入对象性先验，E2V 门控将实体转化为直接的定位证据，三者互补且有清晰的层级递进。
- **实验扎实充分**: 覆盖多个数据集、多种基线，深入的消融和可视化，并跨模型验证结论的通用性，可信度高。
- **可视化出色**: 槽位注意力、门控分数等可视化清晰展示了模型行为，支持了方法内在原理。

### 8. 不足与局限
- **骨干模型单一**: 主要实验基于 Qwen2.5-VL-7B，尽管补充了 InternVL3-2B 的初步分析，但未在更多 MLLM（如 LLaVA 系列）上验证主表性能，普遍性有待进一步确认。
- **视频类型覆盖有限**: 实验仅限于 Charades-STA、QVHighlights、DiDeMo 三个相对标准的 VTG 数据集，未涉及长视频、自拍视角、开放词汇等更具挑战的分布偏移场景，文中也指出了这一局限。
- **知识提炼信号依赖 DINOv2**: EB 蒸馏依赖于 DINOv2 聚类质量，若面对极度域外或抽象视觉内容，该监督的可靠性可能下降；未探讨在没有合适对象性模型时的替代方案。
- **门控所需的概念抽取**: E2V 门控需要离线 LLM 进行主/客体解析，增加了预处理复杂度，且解析错误可能影响性能，尽管文中展示了其鲁棒性。
- **仅聚焦 VTGG**: 方法是为 VTG 任务定制的，迁移到其他视频理解任务的便利性未知。
