## 摘要
视频大语言模型（Video-LLMs）的最新进展使其能够胜任长视频理解任务。然而，现有方法仍面临两个关键局限：证据获取往往依赖于单一搜索意图，而答案生成缺乏有效的视觉反馈机制。为克服这些局限，我们提出 **CoVER**，一个面向长视频理解的综合视觉证据与反思框架。CoVER 使 Video-LLMs 能够通过动态收集基于查询扩展的视觉证据来 **看得更广**，并通过答案特定的有效视觉反馈验证草稿答案来 **想得更深**。这些机制共同将长视频理解从以答案为中心的生成转变为以证据为中心且可视觉验证的推理。实验结果表明，CoVER-7B 大幅超越同等参数规模的模型，并在某些指标上甚至优于最先进的闭源模型。

## Abstract
Recent advances in Video Large Language Models (Video-LLMs) have enabled performance on long-video understanding tasks. However, existing methods still face two key limitations: evidence acquisition often relies on a single search intent, and answer generation lacks an effective visual feedback mechanism. To address these limitations, we propose \textbf{CoVER}, a Comprehensive Visual Evidence and Reflection framework for long-video understanding. CoVER enables Video-LLMs to \textbf{See More} by dynamically gathering query-expanded visual evidence, and \textbf{Think Deeper} by verifying draft answers with effective answer-specific visual feedback. Together, these mechanisms shift long-video understanding from answer-centric generation to evidence-centric and visually verifiable reasoning. Experimental results show that CoVER-7B substantially outperforms models with the same parameter scale and even surpasses state-of-the-art closed-source models on certain metrics.

### 2. 方法论

#### 2.1 核心思想
- **双模块设计**：
  - **查询扩展视觉证据（Query-Expanded Visual Evidence, QVE）**：用模型生成的多个“伪查询（pseudo-queries）”从长视频中检索多样化的细粒度视觉线索，解决单一查询证据覆盖不足的问题。
  - **答案线索引导反思（Answer-Clue Guided Reflection, AGR）**：从草稿答案中提取可验证的视觉线索（clue），再次检索视频证据，判断答案是否得到视觉支持，必要时进行修正，形成闭环推理。

#### 2.2 关键技术与流程
1. **低分辨率全局观察**：先对视频进行均匀采样，获取整体语义上下文，降低计算开销。
2. **CueZoom 工具**：面向文本线索（原始问题、伪查询、答案线索）的视觉证据缩放工具。
   - 计算每一帧与所有文本线索的余弦相似度，取均值作为相关性分数：
     $$s_i = \frac{1}{M}\sum_{j=1}^{M} \text{sim}\big(\phi_t(u_j), \phi_v(f_i)\big)$$
   - 使用**拉普拉斯时间平滑（Laplacian Temporal Smoothing）**去除噪声：
     $$\tilde{s}_i = \sum_{j=-r}^{r} \kappa_j s_{i+j}, \quad \kappa_j \propto \exp(-|j|/h)$$
   - 自适应阈值 $\tau = \mu(\tilde{s}) + \alpha \sigma(\tilde{s})$ 筛选高分帧，合并相邻帧形成连贯的证据片段。
3. **Query-Expanded Visual Evidence (QVE)**：
   - 根据原始问题 $q$ 和全局观察 $V_{\text{global}}$，模型生成至多 3 个伪查询 $P$，与 $q$ 共同作为 CueZoom 的输入，检索高分辨率的证据片段 $C_{qe}$。
   - 基于这些扩展证据生成草稿答案。
4. **Answer-Clue Guided Reflection (AGR)**：
   - 从草稿答案 $\hat{a}$ 中生成答案特定的视觉线索 $c$（描述“若答案正确，视频中应出现何种视觉条件”）。
   - 以 $c$ 作为验证线索调用 CueZoom，检索验证片段。
   - 模型判断草稿答案是否与验证证据一致，保留或修正答案。

#### 2.3 训练范式
- **两阶段监督微调（SFT）**：
  - **Stage-1 (QVE SFT)**：训练模型在证据不足时调用工具并用伪查询检索额外证据，在证据充足时直接生成草稿答案，均需输出答案线索。
  - **Stage-2 (AGR SFT)**：训练模型基于答案线索和验证证据对草稿答案进行验证与修正。**掩蔽错误草稿答案的损失**，避免因错误中间答案误导优化。
- **证据引导的强化学习（GRPO）**：使用组合奖励函数继续优化：
  $$R(\tau) = 0.6 R_{\text{ans}} + 0.25 R_{\text{ver}} + 0.15 R_{\text{fmt}}$$
  - $R_{\text{ans}}$：最终答案正确性。
  - $R_{\text{ver}}$：验证行为奖励，鼓励正确的修正，惩罚无故推翻正确草稿答案、缺少草稿或最终答案等。
  - $R_{\text{fmt}}$：输出格式合规性。

### 6. 主要结论与发现

- CoVER 在同为 7B 参数的模型上显著优于基线 Qwen2.5-VL-7B，在 MLVU、VideoMME、LongVideoBench、LVBench 上分别提升 +3.9%、+0.7% (整体)/+1.1% (Long)、+2.3%、+4.6%。
- 超越多种开源 Agent 型和推理型模型（如 Conan、Video-MTR），部分指标甚至优于闭源 GPT-4o。
- 两阶段 SFT 和 RL 均为有效训练步骤，移除任一部分均会导致性能下降，尤其是 RL 对 MLVU 和 LVBench 增益最大。
- QVE 与 AGR 模块相互补充，移除任一模块均导致性能下降，证明扩展证据获取与答案验证对长视频推理至关重要。

### 7. 优点

- **创新性强**：首次将“视觉反馈反思”引入 Video-LLM 推理，构建**闭环式证据验证流程**，打破开环答案生成的局限。
- **问题切中要害**：直接针对“高答案正确率≠证据正确”这一根本缺陷，提升模型的可信度与可解释性。
- **模块优雅且通用**：CueZoom 工具使用文本-视觉相似度加时间平滑进行无监督检索，无需额外标注，可适配多种模型。
- **训练策略细致**：两阶段 SFT 巧妙掩蔽错误草稿答案的损失，强化学习设计联合奖励函数兼顾答案正确、验证过程和格式，训练信号丰富。
- **实验扎实**：多基准、多模型对比，消融充分，并分析了工具超参数的敏感性，提供了有效性的全面证据。

### 8. 不足与局限

- **查询依赖生成质量**：QVE 的伪查询和 AGR 的答案线索均由模型生成，若生成质量低（歧义、偏见），可能检索到不完整或误导性证据。
- **错误传播风险**：AGR 依赖初始草稿答案，若草稿答案本身错误，可能生成有偏的验证线索，影响修正效果。
- **计算开销**：相比直接均匀采样，CoVER 的多轮工具调用与高分辨率片段检索引入了额外计算成本。
- **适用场景受限**：对于需要整体理解全局叙事的题目，局部缩放证据可能收益有限。
- **实验细节透明性**：未提供伪查询和线索生成失败率的分析，也未深入分析 AGR 修正错误草稿答案的比例与常见失败模式。
- **未见对多语言/多模态**（如音频）的讨论，长视频中的语音信息未被纳入检索。
