## 摘要
从自然经验中学习接地词语含义需要解决婴儿视角记录中的两个模糊性：被命名的所指对象何时出现以及它位于杂乱画面中的何处。在SAYCam风格的数据中，照顾者的语音稀疏且与自我中心视频弱同步，因此单帧对比配对会产生噪声正例，其中目标物体缺失或与干扰物混淆。我们提出BabyMind，一种在稀疏、噪声监督下进行儿童视角对比学习的物体优先偏置。BabyMind使用离线掩码区域接口提取候选物体嵌入，通过跟踪将简短话语中心窗口内的候选关联为轻量级物体文件，并利用原型空间多实例对比目标将话语与物体文件袋对齐。跟踪连贯性和全局物体一致性正则化项稳定学习过程，并将物体文件结构迁移到评估所用的全局帧嵌入中。在SAYCam-S上，BabyMind将Labeled-S的15选1强迫选择准确率相较于CVCL提升了2.6个百分点，并在词汇内分布外基准测试上取得一致增益。代码见https://github.com/sathiiii/BabyMind。

## Abstract
Learning grounded word meaning from natural experience requires resolving two ambiguities in infant-view recordings: when the named referent appears and where it is in a cluttered frame. In SAYCam-style data, caregiver speech is sparse and weakly synchronized with egocentric video, so single-frame contrastive pairing yields noisy positives in which the intended object is absent or entangled with distractors. We propose BabyMind, an object-first bias for child-view contrastive learning under sparse, noisy supervision. BabyMind extracts candidate object embeddings using an offline mask-based region interface, links candidates across a short utterance-centered window into lightweight object files via tracking, and aligns utterances to bags of object files with a prototype-space multiple-instance contrastive objective. Track-coherence and global-object agreement regularizers stabilize learning and transfer object-file structure into the global frame embedding used at evaluation. On SAYCam-S, BabyMind improves Labeled-S 15 forced-choice accuracy by +2.6 points over CVCL and yields consistent gains on in-vocabulary out-of-distribution benchmarks. Code is available at https://github.com/sathiiii/BabyMind.

- **核心问题**：婴儿如何从稀疏、弱同步的感知输入中学会词语与所指对象之间的接地关系。在类似于SAYCam的儿童视角视频中，照顾者的话语间歇出现且与视觉帧的时间对齐很弱，导致两个根本性模糊：

- **现有方法的不足**：基于单帧对比学习的CVCL方法很容易将话语与不含目标物体，或混杂了背景与干扰物的帧配对，产生高噪声正例，尤其对罕见概念更不利。
- **整体含义**：本文提出“物体先于词语”（Objects Before Words）的归纳偏置，主张在学习接地词义时，应先在感知层面形成物体级别的结构化表征，再与语言信号对齐，从而降低噪声并提升对齐质量。

### 2. 论文提出的方法论
- **核心思想**：在CVCL的全局对比学习基础上，增加一条辅助的“物体文件”路径。该路径利用离线生成的分割掩码提取候选物体区域，在一个围绕话语的短时间窗口内将其跟踪为轻量物体文件，并通过原型空间的多实例对比学习（MIL）将话语与整个物体文件袋对齐，同时用两个正则项将物体结构注入全局嵌入。

- **关键技术细节与流程**：
  - **物体候选与嵌入**：
    - 对每帧使用离线自动掩码生成（AMG，基于SAM）得到区域掩码，下采样到特征图分辨率后进行掩码平均池化，得到区域描述符。
    - 区域描述符经投影头映射并归一化，得到候选物体嵌入$\boldsymbol{o}_{i,m,r}$。
    - 若掩码失效或缺失，自动回退为补丁候选（选取特征图上显著位置的特征），保证每帧都有有效候选。
  - **短窗口物体文件构建**：
    - 在一个话语中心的$M$帧窗口内，按时间顺序贪婪追踪候选物体：计算与已有轨道的余弦相似度，高于阈值（0.55）则合并，否则创建新轨道。
    - 轨道嵌入定义为所包含候选物体嵌入的$\ell_2$归一化均值。引入可学习的空嵌入$\tilde{\boldsymbol{o}}_{\varnothing}$表示无接地对象。
  - **原型记忆与MIL对齐**：
    - 维护一个原型码本$\mathbf{P}$（$K=64$个可学习原型向量），将文本嵌入$\boldsymbol{u}_i$和物体文件嵌入$\tilde{\boldsymbol{o}}_{i,r}$分别计算软分配$\boldsymbol{q}^{\text{txt}}_i$和$\boldsymbol{q}^{\text{trk}}_{i,r}$。
    - 在原型空间执行MIL：对话语-物体文件袋计算相似性得分，采用掩码log-sum-exp池化聚合为$S^{\text{mil}}_{ij}$，再施加对称对比损失$\mathcal{L}_{\text{mil}}$。
  - **正则化器**：
    - **跟踪连贯性**（$\mathcal{L}_{\text{coh}}$）：强制同一轨道内各帧候选的原型分配与轨道整体分配一致，使用KL散度并配合停止梯度。
    - **全局物体一致性**（$\mathcal{L}_{\text{go}}$）：从与文本最匹配的轨道出发，用原型混合物重构出一个嵌入$\hat{\boldsymbol{o}}_i$，并最小化该嵌入与全局帧嵌入$\boldsymbol{v}_i$之间的余弦距离，从而将物体结构迁移到评估所用的全局表示中。
  - **总体目标**：
    \[
    \mathcal{L} = \lambda_{\text{glob}}\mathcal{L}_{\text{glob}} + \lambda_{\text{MIL}}\mathcal{L}_{\text{mil}} + \lambda_{\text{coh}}\mathcal{L}_{\text{coh}} + \lambda_{\text{go}}\mathcal{L}_{\text{go}}
    \]
    其中$\lambda_{\text{glob}}=1.0$，$\lambda_{\text{MIL}}=0.10$，$\lambda_{\text{coh}}=0.05$，$\lambda_{\text{go}}=0.05$，MIL温度$\tau_{\text{MIL}}=0.05$。

### 7. 优点
- **独特的归纳偏置设计**：明确将认知科学中的“物体文件”概念与现代对比学习框架结合，在架构层面注入感知结构，而非单纯依赖数据增强。
- **巧妙解决时空模糊**：利用短窗口跟踪和MIL，有效放宽了严格单帧配对要求，使模型能容忍时间错位和掩码缺失。

- **可解释性与诊断**：通过原型记忆、跟踪可视化和消融实验，方法内部的行为高度可解释。
