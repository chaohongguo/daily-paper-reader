## 摘要
时空视频定位旨在根据文本查询定位目标管。尽管近期的方法取得了显著成功，它们主要关注高质量(HQ)输入，忽略了现实场景中普遍存在的低质量(LQ)视频。虽然像LoRA这样的调优方法能够适应退化输入，但它们不可避免地会破坏预训练知识。为解决这一问题，我们提出了零空间调优(NST)。该框架利用了这样一个几何性质：在冻结权重的零空间中向层输入添加向量不会影响输出。利用这一点，NST将可学习残差注入输入特征，这些残差可以选择性地对预训练骨干网络不可见。具体来说，NST结合了质量自适应单元和双空间重参数化来合成这些残差，通过将针对HQ输入的成分限制在零空间，而将针对LQ输入的恢复成分引导到非零空间。由于冻结的权重会消除零空间成分，我们有效地纠正了退化输入，同时为HQ输入保留了预训练知识。大量实验表明，NST在我们的混合质量基准上优于最先进的方法。

## Abstract
Spatio-Temporal Video Grounding aims to localize object tubes based on textual queries. While recent methods have achieved remarkable success, they mainly focus on high-quality(HQ) inputs, neglecting the widespread presence of low-quality(LQ) videos in real-world scenarios. Although tuning methods like LoRA can adapt to degraded inputs, they inevitably disrupt pre-trained knowledge. To address this, we propose Null-Space Tuning (NST). This framework exploits the geometric property that adding vectors within the null-space of frozen weights to the layer input does not affect the output. Leveraging this, NST injects learnable residuals into input features that can be selectively invisible to the pre-trained backbone. Specifically, NST combines the Quality-Adaptive Unit and Dual-Space Reparameterization to synthesize these residuals by confining components for HQ inputs to the null-space, while directing restoration components for LQ inputs to the non-null space. As the frozen weights eliminate null-space components, we effectively rectify degraded inputs while preserving pre-trained knowledge for HQ inputs. Extensive experiments show that NST outperforms state-of-the-art methods on our Mixed-Quality benchmark.

### 1. 论文核心问题与研究动机
- **问题背景**：时空视频定位（Spatio-Temporal Video Grounding, STVG）旨在依据自然语言查询在未剪辑视频中定位目标物体的时空管（tube）。现有先进方法（如基于 DETR 架构的模型）主要在高质量（HQ）视频上取得了优异性能，却忽略了真实场景中由于运动模糊、失焦、低光照和压缩等退化导致的低质量（LQ）视频。
- **核心矛盾**：若直接使用参数高效微调（PEFT）方法（如 LoRA）来适应退化输入，由于 HQ 与 LQ 特征共享同一投影空间，调整参数以修复 LQ 数据会不可避免地干扰甚至灾难性遗忘 HQ 上的预训练知识，形成“修复—遗忘”两难。
- **论文目标**：提出一种在适配低质量输入的同时严格保持预训练 HQ 性能的鲁棒调优框架。

### 2. 方法核心思想与技术细节
- **总体框架：Null-Space Tuning (NST)**  
  冻结骨干权重，仅在解码器中低秩层注入两个即插即用模块，通过几何手段使注入的残差对不同质量输入呈现选择性可见性。
  
- **关键几何性质**：对冻结权重 \(W\) 做 SVD 分解 \(W = U\Sigma V^\top\)，将输入空间分解为行空间（非零空间，\(V_{\text{row}}\)）和零空间（\(V_{\text{null}}\)）。零空间内的任何向量经 \(W\) 作用后输出为零。NST 利用这一性质设计残差：
  - 对 HQ 输入，残差 \(\delta\) 被约束在 \(V_{\text{null}}\)，使得 \(W(F+\delta)=WF\)，严格保持原输出。
  - 对 LQ 输入，残差 \(\delta\) 被引导至 \(V_{\text{row}}\)，使得 \(W(F+\delta)=WF+W\delta\)，实现特征修复。

- **质量自适应单元 (QAU)**  
  构造多模态参考库 \(M = [\text{Proj}(\bar{F}); \text{Proj}(F_{\text{text}})]\)，其中 \(\bar{F}\) 为全局视觉上下文（时间池化），\(F_{\text{text}}\) 为文本特征。以退化帧特征 \(F\) 为查询，通过多头交叉注意力从参考库检索缺失语义，得到检索特征 \(H\)。随后经两个并行的 MLP 生成双域投影系数：
  \[
  \vec{\alpha} = \text{MLP}_{\text{null}}(H), \quad \vec{\beta} = \text{MLP}_{\text{row}}(H)
  \]
  \(\vec{\beta}\) 携带恢复信息（进入行空间），\(\vec{\alpha}\) 收集可能污染 HQ 的噪声（进入零空间）。

- **双空间重参数化**  
  由语义间隙 \(H\) 经轻量门控 \(s = \sigma(\text{MLP}_{\text{gate}}(\text{AvgPool}(H)))\) 动态控制：
  \[
  \delta = s \cdot (\vec{\beta} V_{\text{row}}^\top) + (1-s) \cdot (\vec{\alpha} V_{\text{null}}^\top)
  \]
  最终特征 \(\hat{F} = F + \delta\)。HQ 时 \(s \to 0\)，残差几乎全在零空间被消除；LQ 时 \(s \to 1\)，恢复分量通过行空间生效。

- **优化目标**  
  总损失 \(L_{\text{total}} = L_{\text{STVG}} + \lambda_{\text{gate}} L_{\text{gate}} + \lambda_{\text{reg}} L_{\text{reg}}\)，其中 \(L_{\text{gate}}\) 监督门控 \(s\)，\(L_{\text{reg}}\) 根据标签抑制非对应空间分量，防止 HQ 行空间泄漏和 LQ 零空间爆炸。

### 7. 优点
- **创新性强**：首次将冻结权重的零空间用于调优，几何上解耦 HQ 保持与 LQ 修复，避免灾难性遗忘。
- **参数高效**：仅插入轻量 QAU 和双空间投影，冻结原始骨干，引入参数量少。
- **实例级自适应**：通过门控 \(s\) 实现逐样本动态路由，优于传统静态注入 PEFT。
- **实验扎实**：构建了真实感退化管道和混合质量基准，提出 WAS 指标，并进行了详尽的消融和层选择分析。
- **理论与工程兼顾**：利用 SVD 预计算缓存，推理开销极小。

### 8. 不足与局限
- **适用层限制**：方法依赖于线性投影层的几何冗余，仅对具有显著零空间的低秩层有效，对高秩全秩层无效，限制了应用范围。
- **退化类型模拟**：退化管道虽包含常见的模糊和低光，但未涵盖所有真实场景（如遮挡、恶劣天气），实验的可迁移性有待进一步验证。
- **计算与存储开销**：SVD 分解需预计算并缓存，对大规模模型可能增加离线处理负担；训练时需额外监督标签 \(y_q\)，在无干净标签的部署场景中可能需要弱监督或自监督设计。
- **任务普适性未验证**：目前仅在 STVG 上验证，在其他视频理解任务（如动作识别、VQA）中的表现尚未讨论。
