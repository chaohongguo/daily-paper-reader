## 摘要
WALL-WM 是一个世界行动模型，它将视频-行动学习从以块为中心的优化转变为基于事件的视觉-语言-行动预训练，以语义连贯的行动事件作为学习的原子单元。现有世界行动模型通常从多模态或视频基础模型初始化，然后以当前观测和指令为条件优化固定长度的动作块。这种做法虽然方便，但以块为中心的公式会造成根本的粒度失配。语言描述语义目标和事件，视觉通过连续场景动态演变，而行动在控制级时间尺度上运作；强制三者进入同一固定长度预测窗口，会将视觉-语言-行动训练变成短视域关联拟合。WALL-WM 通过围绕语义事件组织监督和数据来解决这一失配问题。具体来说，它将基于事件的视觉-语言-行动预训练与由事件级描述和集群平衡采样构建的数据生态系统相结合，从而能够在多样化行为、场景和任务结构上实现可扩展学习。基于同一事件预训练的主干，WALL-WM 支持两种互补的推理模式。事件模式接收下一事件描述，并支持可变长度的执行块；而统一模式则使用配备阶梯解码的视觉-语言模型来调节常规固定长度块推理，同时保持梯度连续的视觉-语言-行动通路。结合基于 Muon 优化器的大规模预训练基础设施，WALL-WM 为通用世界行动模型提供了一套实用的扩展方案。实验表明，WALL-WM 在语言、场景和任务上泛化广泛，在大规模真实世界泛化评估中取得了最先进的性能。

## Abstract
WALL-WM is a World Action Model that shifts video-action learning from chunk-centric optimization to event-grounded Vision-Language-Action pretraining, using semantically coherent action events as the atomic unit of learning. Existing WAMs commonly initialize from multimodal or video foundation models and then optimize fixed-length action chunks conditioned directly on the current observation and instruction. Although convenient, this chunk-centric formulation creates a fundamental granularity mismatch. Language describes semantic goals and events, vision evolves through continuous scene dynamics, and actions operate at control-level timescales; forcing all three into the same fixed-length prediction window turns VLA training into short-horizon correlation fitting. WALL-WM addresses this mismatch by organizing both supervision and data around semantic events. Specifically, it pairs event-grounded VLA pretraining with a data ecosystem built from event-level captions and cluster-balanced sampling, enabling scalable learning over diverse behaviors, scenes, and task structures. From the same event-pretrained backbone, WALL-WM supports two complementary inference modes. The event mode consumes next-event descriptions and enables variable-length execution chunks, while the unified mode uses a VLM with Staircase Decoding to condition conventional fixed-length chunk inference while preserving a gradient-continuous VLA path. Together with Muon-optimizer-based large-scale pretraining infrastructure, WALL-WM provides a practical scale-up recipe for general-purpose WAMs. Experiments show that WALL-WM generalizes broadly across language, scenes, and tasks, achieving state-of-the-art performance in large-scale real-world generalization evaluation.

### **2. 方法论：WALL-WM 的核心思想与关键技术**

WALL-WM的核心思想是**“在自然的连接处切割世界（Carve Nature at its Joints）”**，即将**语义连贯、可执行的动作事件**作为学习的原子单元，而非固定长度的时钟切片。

*   **核心思想：事件接地（Event-Grounded）预训练**
    *   将每个完整的语义行为（如“拿起杯子”、“移动到目标位置”）视为一个基本训练单元。
    *   每个单元包含**事件描述、事件视频和事件动作**的三元组，三者天然对齐，消除了粒度和语义上的模糊性。
    *   **两大推理模式**：
        *   **事件模式**：根据“下一事件”的描述，生成并执行一个可变长度的视频-动作片段，执行完毕后再规划下一个事件，形成一个闭环的事件级展开。
        *   **统一模式**：通过一个带有**阶梯解码**的视觉大模型（VLM），将全局指令转化为潜在的、连续的事件结构化推理状态，以此驱动传统的固定长度块预测，在保留梯度连续性的同时引入了事件级的语义指导。

*   **关键技术细节：**
    1.  **层耦合的视频-行动去噪器**：
        *   继承并扩展了Wan系列文本到视频生成模型作为视频塔，并新增一个随机初始化的动作DiT（Diffusion Transformer）作为动作塔。
        *   双塔在每一层进行单向耦合：动作塔的每一层通过交叉注意力机制读取同一层视频塔的输出特征，从而将视觉动态自然地融入行动预测。
        *   **多视角适配**：通过“交叉视角自注意力”、“相机RoPE”和“交叉视角几何掩码”等技术，将从互联网视频学习的单视角先验适配到多视角、多具身机器人场景。
    2.  **视频-动作时间对齐**：
        *   **事件中心窗口**：预训练和事件推理时使用，通过可学习的`Eτ`位置编码，为视频帧和对应的动作步建立精准的对齐关系。
        *   **观测中心窗口**：统一推理时使用，引入历史帧和绝对窗口索引`Eabs`编码，使模型能理解当前块在完整任务中的位置。
    3.  **动作流匹配与时间步映射**：
        *   采用流匹配（Flow Matching）目标训练动作塔。在默认的大规模训练中，使用**非对称的1-to-N映射**：将视频塔的前向过程固定在一个中间噪声锚点`s*`，让动作塔在该锚点提供的稳定视觉特征上完成所有去噪步的学习，这样既利用了清晰的视觉结构，又保证了动作预测对噪声的鲁棒性。
    4.  **阶梯解码（Staircase Decoding）**：
        *   一种高效的并行潜在思维链生成方法。它将一个 Transformer 模型在某个**中继深度**处分割。
        *   **底层**计算共享的视觉-语言基础特征，**高层**则并行生成多个连续的潜在推理状态。
        *   这避免了传统自回归推理的串行瓶颈，大幅度降低了推理延迟。

*   **训练流程：**
    1.  **视频预训练**：在事件级数据上，冻结T5文本编码器，仅训练多视角视频DiT。
    2.  **动作预训练**：冻结视频DiT，仅训练动作DiT，学习从视频特征到动作轨迹的映射。
    3.  **VLM文本调节器训练**：训练一个轻量级投影头，将冻结的Qwen-9B VLM的隐藏状态对齐到DiT的T5文本特征空间，并预测下一事件和剩余时间。
    4.  **阶梯蒸馏**：冻结主VLM，仅训练阶梯推理分支和前缀投影器，通过重构文本思维链来学习紧凑的潜在推理状态。
    5.  **下一块适配**：可选阶段，将事件预训练的主干微调到观测中心窗口的固定块预测范式上。

*   **解决了粒度失配，实现了真正的多模态对齐**：模型生成的视频在运动质量、物理合理性和交互质量上显著优于通用的视频生成模型，说明了将事件作为对齐锚点的有效性。
*   **结合大语言模型推理能有效解决复杂任务**：事件模式下，VLM将高层指令分解为可执行的事件描述，强化了模型在推理和泛化任务上的表现。
*   **灵巧操作仍然是挑战**：所有模型在灵巧操作套件中的绝对分数都不高，表明精细接触控制依然是该领域的难点。
*   **跨视角建模的重要性**：消融实验表明，移除视角交互自注意力会显著降低推理和泛化性能，证明了多视角特征融合的价值。

### **7. 优点与亮点**

*   **方法论创新性强**：直指当前范式痛点，提出了以语义事件为原子单位的根本性解决方案，思想深刻。
*   **设计逻辑连贯且周全**：从根本性问题出发，推导出设计原则，然后将原则落实到数据处理、模型架构、训练和推理的每个环节，构建了一个完整的闭环系统。
*   **工程实现扎实**：在数据构建，多视角适配，分布式优化器量化，低成本部署等软硬件工程上均有深入探索和贡献。
*   **评测体系合理**：采用“任务进度”而非简单的“成功率”，并坦承自身环境优势，体现了严谨和诚实的科研态度。
*   **性能全面领先**：在视频生成和真实机器人操作两大任务上，相较于多个强大基线取得了实质性的性能提升。

### **8. 不足与局限**

*   **庞大的系统工程依赖**：该方法的成功依赖于大规模、高质量的事件级标注数据，复杂的多阶段训练流程和定制化的工程优化，复现门槛极高。
*   **评测集可能存在偏差**：真实机器人的评估仅在与训练数据高度适配的自研平台上进行，其在低成本、非结构性环境以及更广泛第三方硬件上的泛化能力仍需外部验证。
*   **与最大规模最强基线的对比不足**：虽然对比了π0.5，但对比的基线数量有限，且可能未包含同时期其他更强大的模型，对比的广泛性有待加强。
*   **事件边界依赖性**：当前方法严重依赖人工标注的事件边界和分层标题，作者也承认了这是一个局限，并提出了未来自监督学习的探索方向。
*   **灵巧操作性能不佳**：即使在论文提出的最强模型下，精细操作任务的绝对完成度依然很低，说明该框架在接触级动力学处理上仍有较大提升空间。
