## 摘要
微手势分析因能从细微的身体动作推断自发情感而日益受到关注。微手势在线识别，即在未剪辑视频中定位并分类每个手势实例，是第四届EI-MiGA-IJCAI竞赛的核心任务之一。与典型的时间动作检测相比，微手势识别强调动作的定位与分类，要求模型输出每个微手势的起始时间、结束时间和类别。此外，由于微手势具有高度自发性，仅依赖单一模态难以捕捉完整、准确的多模态线索。本文中，我们提出DyFADet+，将DyFADet扩展为双流RGB-骨架框架。在我们的模型中，两种模态被投影到共享的多尺度时间嵌入中，并通过一个门控残差模块进行融合，该模块自适应地将骨架运动注入RGB表示，而非采用简单的拼接。最终，这些融合特征由动态TAD头解码，用于在线分类和边界回归。在SMG数据集上，我们的方法取得了40.88的F1分数，在微手势在线识别赛道中位列第二。

## Abstract
Micro-gesture analysis attracts increasing attention for inferring spontaneous emotion from subtle body movements. Micro-gesture online recognition, which localizes and classifies each gesture instance in untrimmed videos, is a core task in the 4th EI-MiGA-IJCAI Challenge. Compared with typical temporal action detection, MGR emphasizes the localization and classification of actions, requiring the model to output the start time, end time, and category of each micro-gesture. Moreover, since micro-gestures are highly spontaneous, relying solely on a single modality makes it difficult to capture the complete and accurate multi-modal cues. In this work, we propose DyFADet+, which extends DyFADet into a dual-stream RGB-skeleton framework. In our model, both modalities are projected into shared multi-scale temporal embeddings and fused through a gated residual module, which adaptively injects skeleton motion into the RGB representation rather than using naive concatenation. Finally, these fused features are decoded by a Dynamic TAD head for online classification and boundary regression. On the SMG dataset, our method achieves an F1 score of 40.88, ranking 2nd in the Micro-gesture Online Recognition track.

### 2. 论文提出的方法论：DyFADet+

*   **核心思想：** 将成熟的单流时间动作检测器DyFADet扩展为双流RGB-骨架框架，通过一种新颖的**门控残差融合机制**，让骨架的运动信息有选择性地、以残差的形式注入并增强RGB主特征流，而非进行简单的特征叠加或拼接，从而在保留RGB外观判别力的同时，强化运动敏感区域的表征。

*   **关键技术细节与算法流程：**
    1.  **双流特征编码：**
        *   **RGB流**：使用预训练模型（如VideoMAEv2-g）提取特征，并沿用DyFADet的动态编码器（DynE blocks）对序列进行多尺度时间下采样，构建多尺度特征金字塔 `{F(ℓ)_r}`。
        *   **骨架流**：将预提取的骨架特征通过一系列1D卷积块，同样进行多级时间下采样，构建与RGB流在时间维度上严格对齐的多尺度特征金字塔 `{F(ℓ)_s}`，长度不对齐时采用最近邻插值。

    2.  **门控残差融合 (Gated Residual Fusion)：这是本方法的核心。对于金字塔的每一层 ℓ：**
        *   **骨架特征适配**：一个轻量级的适配器 `ψℓ`（两层带GELU激活的时序卷积）将骨架特征 `F(ℓ)_s` 映射到与RGB特征 `F(ℓ)_r` 相同的通道空间。
        *   **门控信号生成**：将RGB和骨架特征沿通道维度拼接，输入一个门控函数 `ϕℓ`（一个1x1时间卷积 + Sigmoid激活函数）生成空间-时序维度的门控矩阵 `G(ℓ)`。该门控矩阵的值在0到1之间，代表了在当前时刻和特征通道上对骨架信息的可信度/采纳程度。
        *   **残差融合公式**：
            `G(ℓ) = σ [ ϕℓ([F(ℓ)_r, F(ℓ)_s]) ]`
            `F(ℓ)_f = F(ℓ)_r + α · G(ℓ) ⊙ ψℓ(F(ℓ)_s)`
            其中，`α = 0.5` 是残差缩放因子，`⊙` 表示逐元素乘法。该设计使得骨架信息 (`ψℓ(F(ℓ)_s)`) 只有在门控矩阵 `G(ℓ)` 激活时才被有选择地加到RGB特征上，实现了“运动增强外观”的目标。初始化时，门控卷积的偏置被设为-2，使初始门控值接近0，确保训练初期以稳定的RGB路径为主。

    3.  **动态TAD头 (Dynamic TAD Head)**：融合后的多尺度金字塔特征 `{F(ℓ)_f}` 直接输入到DyFADet原有的动态检测头中。
        *   **密集预测**：在时间轴上以预设的步长`{1, 2, 4, 8, 16, 32}`生成密集点，并为其分配不同的尺度感知回归范围，以检测不同时长的微手势。
        *   **双分支输出**：
            *   分类分支：1D卷积 + Sigmoid，预测每个点属于17个类别的概率。
            *   回归分支：1D卷积 + ReLU，预测该点到动作开始和结束边界的距离。

    4.  **损失函数：** 采用Focal Loss用于分类，DIoU Loss用于边界回归，两者联合进行端到端训练。

*   **核心结论：** 提出的DyFADet+框架通过门控残差融合机制，能够有效利用骨架运动信息来增强RGB外观特征，从而显著提升微手势在线识别的性能。
